# Outrage

**Outrage** (formerly Ranxware) is an upcoming Roblox exploit utility primarily providing overpowered advantages while being undetectable.

Outrage also includes software tools, like custom animation saving and remote call hooking.

## Upcoming Software
* **Outrage Beta** - The main script giving users advantages against opponents
* **Macrohook** - The tool utility capable of hooking remote calls, as well as function hooking and table changing
* **AnimSaver** - Saves custom animations into files (instance file or json data)
